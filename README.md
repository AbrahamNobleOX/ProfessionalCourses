# C#, .NET, MVC & Entity Framework using N-Tier Architechure - The Setup

Configuration Steps

1. Run the SQL database script to create the database and tables.
2. Connect the database in the server explorer.									
3. Delete the connection in the PostModel.edmx file by highlighting the connection string and pressing delete.
4. Right click on the PostModel.edmx file and select Update Model from Database.	
4. Update the connection string in the web.config file.
5. Run the application.
